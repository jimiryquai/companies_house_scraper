# Snov.io Integration - Lite Summary

Enrich Companies House officer data with email addresses and company domains using Snov.io API, implementing a domain-first discovery strategy through our existing queue system.

## Key Points
- Domain discovery from company names using Snov.io Domain Search API
- Officer email finding using discovered domains and officer names
- Credit tracking and management system with cost controls and monitoring
- Database caching to minimize API usage and improve performance
- Queue-based processing with HIGH priority for efficient batch operations
- Webhook handling for bulk operation status updates and results processing
